애플리케이션 정보

- 애플리케이션 명: SSAFI
- 버전: 1.0.0
- 배포 일자: 2023-10-06

----------------------------------------------------------------------------------------------

배포 전 확인 사항

1. 환경변수 설정
- DB_CONNECTION_STRING 환경 변수가 올바르게 설정되어 있는지 확인
- API_KEY 값이 최신인지 확인

2. 종속성 체크
- 필요한 모든 패키지가 requirements.txt 혹은 package.json에 명시되어 있는지 확인

3. 데이터베이스 백업
- 배포 전에 현재 데이터베이스의 백업을 수행

----------------------------------------------------------------------------------------------

배포 절차

1. 코드 업데이트
- git pull origin main 명령어로 최신 코드를 받아옵니다.

2. 패키지 설치
- npm install 혹은 pip install -r requirements.txt로 종속성을 설치합니다.

3. 데이터 마이그레이션
- 필요한 경우 python manage.py migrate 등의 명령어로 데이터베이스 마이그레이션을 수행합니다.

4. 테스트 실행
- npm test 혹은 python manage.py test 등의 명령어로 모든 테스트를 실행하여 문제가 없는지 확인합니다.

5. 애플리케이션 재시작
- systemctl restart ssafi 명령어로 애플리케이션을 재시작합니다.

----------------------------------------------------------------------------------------------

배포 관련

- redis, mysql 등 docker 내 설치
- application-local.yml, application-dev.yml 등 환경 설정 파일은 수동 배포, 나머지 값들은 git에 등록된 dev 브랜치를 읽어 자동 배포.

----------------------------------------------------------------------------------------------

주의 사항

- API 엔드포인트 변경: 이번 배포에서 엔드포인트가 /api 를 붙이는 형식으로 변경되었습니다.

----------------------------------------------------------------------------------------------

롤백 절차

1. 애플리케이션 재시작: systemctl restart ssafi

2. 필요한 경우 데이터베이스 롤백 수행
